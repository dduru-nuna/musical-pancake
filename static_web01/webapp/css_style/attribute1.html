글자 관련 스타일 속성

font-size: 글자 크기를 변경하는 스타일 속성

font-weight: 글자 두께를 변경하는 스타일 속성

font-style: 글자에 기울임 스타일을 적용하기 위한 속성

font-family: 글자 폰트를 적용하기 위한 속성

color: 글자 색을 적용하기 위한 속성

text-decoration: 글자에 밑줄과 같은 스타일을 적용하기 위한 속성

링크 이동

text-shadow: 글자에 그림자 스타일을 적용하기 위한 속성

text-align: 글자 정렬을 위해 사용하는 속성으로 left 값을 사용하면 왼쪽으로 정렬한다.

text-align: 글자 정렬을 위해 사용하는 속성으로 right 값을 사용하면 오른쪽으로 정렬한다.

text-align: 글자 정렬을 위해 사용하는 속성으로 center 값을 사용하면 가운데로 정렬한다.

text-align: 글자 정렬을 위해 사용하는 속성으로 justify 값을 사용하면 양쪽맞춤으로 정렬한다.

line-height: 글자간의 줄 간격을 조절하기 위해 사용하는 속성

letter-spacing: 글자 간격을 조정하기 위해 사용하는 속성

word-spacing: 단어 간격을 조정하기 위해 사용하는 속성

white-space: normal;:공백 처리와 관련된 기능을 제공하는 스타일 기본 공백처리 방식으로 여러 공백이 있어도 하나의 공백만을 반영한다.

white-space: nowrap;:여러 공백이 있어도 하나의 공백만을 반영하고 줄 바꿈이 필요해도 바꾸지 않고 한 줄로 표기한다.

white-space: pre;:여러 공백을 그대로 반영하여 출력한다. 줄 바꿈도 입력한 내용대로 들어간다.

white-space: pre-line;: 여러 공백을 그대로 반영하지 않으며, 줄 바꿈은 자동으로 반영하여 표기한다.(입력한 내용대로 줄바꿈도 반영된다.)

white-space: pre-wrap: 여러 공백을 그대로 반영하며, 줄 바꿈도 자동으로 반영하여 표기한다.(입력한 내용대로 줄바꿈도 반영된다.)

text-overflow: clip;: 지정된 영역을 벗어나는 텍스트 글자에 대한 스타일 지정 clip 속성값은 벗어난 텍스트 글자에 대해 잘라내기 처리를 한다.

text-overflow: ellipsis;: 지정된 영역을 벗어나는 텍스트 글자에 대한 스타일 지정 ellipsis 속성값은 벗어난 텍스트 글자에 대해 말줄임표 처리를 한다.
